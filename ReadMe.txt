How to:
1) Host project on Apache Tomcat localhost server.
2) Open oneShop.html file to run the application on localhost:8080 port.

Technologies Used: HTML5, CSS3, Bootstrap, jQuery, AngularJS, JSON.
Tools Used: 	   Notepad++, Apache Tomcat 8.0.

Application Details:
1) Responsive and interactive E-Commerce user interface, with client-side MVC architecture.
2) Designed based on Mobile-first approach considering Google’s Material Design guidelines.
3) Data is communicated using JSON object array and website hosted on Apache Tomcat local server.
4) Ajax is implemented for category filter and search operation.
